# Redux Toolkit example with a simple shopping cart

## Check demo here:

[Netlify Link](https://rtk-simple-cart.netlify.app)

## Highlights

- Folder structure

  - features/...slice.js
  - app/store.js

- 4 reducers
  - add to cart
  - get cart's total
  - increase quantity of an item
  - decrease quantity of an item
